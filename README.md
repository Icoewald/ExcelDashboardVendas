# Descrição

Dashboard de vendas de base de dados fictícia de assinaturas XBOX.
Desenvolvido para desafio de curso Dashboard em Excel da DIO.

## Utilização
Com basease de dados incluída no arquivo, o dashboard tem por objetivo responder a 4 perguntas de negócio:

1) Faturamento total de vendas por plano: Ultimate, Core, Standard
2) Faturamento total de vendas EA Play Season Pass e Minecraft Season Pass
3) Histórico de novas assinaturas e total de assinantes por mês
4) Distribuição de assinantes por Subscription Type
